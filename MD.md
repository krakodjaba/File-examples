## 14. Вставка диаграмм (Mermaid)

### Граф связей (пример OSINT)

```mermaid
graph LR
    A[Объект] --> B[Соцсети]
    A --> C[Домены]
    C --> D[WHOIS]
    B --> E[LinkedIn]
    B --> F[Telegram]
```

### Таймлайн

```mermaid
timeline
    title Timeline расследования
    2023-01 : Начало наблюдения
    2023-02 : Активность в соцсетях
    2023-03 : Регистрация домена
```

### Блок-схема

```mermaid
flowchart TD
    Start --> Collect[Сбор данных]
    Collect --> Analyze[Анализ]
    Analyze --> Report[Отчёт]
    Report --> End
```
